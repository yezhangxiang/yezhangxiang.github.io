常用的距离的定义方法

发表于 2015-03-16 | 分类于 资料整理

在上一篇文章中整理了常用的聚类算法，应用聚类算法有个前提要定义各个点之间的距离，常用的距离定义在matlab的pdist参数选择中有：

Metric Description
‘euclidean’ Euclidean distance (default).
‘seuclidean’ Standardized Euclidean distance. Each coordinate difference between rows in X is scaled by dividing by the corresponding element of the standard deviation S=nanstd(X). To specify another value for S, use D=pdist(X,’seuclidean’,S).
‘cityblock’ City block metric.
‘minkowski’ Minkowski distance. The default exponent is 2. To specify a different exponent, use D = pdist(X,’minkowski’,P), where P is a scalar positive value of the exponent.
‘chebychev’ Chebychev distance (maximum coordinate difference).
‘mahalanobis’ Mahalanobis distance, using the sample covariance of X as computed by nancov. To compute the distance with a different covariance, use D = pdist(X,’mahalanobis’,C), where the matrix C is symmetric and positive definite.
‘cosine’ One minus the cosine of the included angle between points (treated as vectors).
‘correlation’ One minus the sample correlation between points (treated as sequences of values).
‘spearman’ One minus the sample Spearman’s rank correlation between observations (treated as sequences of values).
‘hamming’ Hamming distance, which is the percentage of coordinates that differ.
‘jaccard’ One minus the Jaccard coefficient, which is the percentage of nonzero coordinates that differ.

给定$m*n$的矩阵$X$，表示n维空间中的m个点，那么向量$x_s$和 $x_t$之间的距离可定义为：

1. 欧氏距离(Euclidean distance)

$$ d^2_{st}=(x_s-x_t)(x_s-x_t)′$$
这也是最常用的距离定义，就不举例了。

2. 标准化欧氏距离 (Standardized Euclidean distance )

$$ d^2_{st}=(x_s-x_t)V^{-1}(x_s-x_t)′$$
where V is the n-by-n diagonal matrix whose jth diagonal element is $S(j)^2$, where S is the vector of standard deviations.
标准化欧氏距离是针对简单欧氏距离的缺点而作的一种改进方案。如果各个维度上分布相差巨大，比如维度d1分布在(-10000, 1000)，而d2分布在(-1, 1)，二者都满足正态分布，那么d1对距离的贡献是d2的1000倍！
标准欧氏距离的思路：既然数据各维分量的分布不一样，好吧！那我先将各个分量都“标准化”到均值、方差相等。

3. 曼哈顿距离(city block distance)

$$ d_{st}=\sum^n_{j=1}|x_{sj}-x_{tj}|$$
city block距离，也叫manhattan距离，在matrix67的一篇博客出租车几何学中有一些有趣的介绍。不过平时好像没怎么用到过。

4. 闵可夫斯基距离(Minkowski distance)

$$ d_{st}=\sqrt[p]{\sum^n_{j=1}|x_{sj}-x_{tj}|^p}$$
即$L^p$度量，p=2即为欧氏距离，而p=1时则为曼哈顿距离。当p取无穷时的极限情况下，可以得到切比雪夫距离。除了学泛函分析时用过，平常好像没什么场景会用到。

5. 切比雪夫距离(chebychev distance)

$$ d_{st}=\max_j|{x_{sj}-x_{tj}}|$$
若将国际象棋棋盘放在二维直角座标系中，格子的边长定义为1，座标的x轴及y轴和棋盘方格平行，原点恰好落在某一格的中心点，则王从一个位置走到其他位置需要的步数恰为二个位置的切比雪夫距离，因此切比雪夫距离也称为棋盘距离。

6. 马氏距离(mahalanobis distance)

$$ d^2_{st}=(x_s-x_t)C^{-1}(x_s-x_t)′$$
where C is the covariance matrix.
若协方差矩阵是单位矩阵（各个样本向量之间独立同分布），也就是欧氏距离了。
若协方差矩阵是对角矩阵，公式变成了标准化欧氏距离。
马氏距离的优缺点：量纲无关，排除变量之间的相关性的干扰。

7. 夹角余弦距离(cosine distance)

$$d_{st}=1-\frac{x_sx^′_t}{\sqrt{(x_sx^′_s)(x_tx^′_t)}}$$
余弦距离比较有名的应用当属吴军的数学之美系的余弦定理和新闻的分类，不过为什么选用余弦距离而不用其他距离并没有说明原因。

8. 相关距离(correlation distance)

$$d_{st}=1-\frac{(x_s-\bar{x}_s)(x_t-\bar{x}_t)’}{\sqrt{((x_s-\bar{x}_s)(x_s-\bar{x}_s)’)}\sqrt{((x_t-\bar{x}_t)(x_t-\bar{x}_t)’)}}$$
where $\bar{x}_s = \frac{1}{n}\sum_{j} x_{sj} $ and $ \bar{x}_t=\frac{1}{n}\sum_{j}x_{tj}$

9. spearman相关系数(spearman distance)

$$d_{st}=1-\frac{(r_s-\bar{r}_s)(r_t-\bar{r}_t)’}{\sqrt{((r_s-\bar{r}_s)(r_s-\bar{r}_s)’)}\sqrt{((r_t-\bar{r}_t)(r_t-\bar{r}_t)’)}}$$
其中$r$是秩相关系数，又称等级相关系数，介于-1到1之间，$r<0$为负相关，$r>0$为正相关.且有
$\bar{r}_s=\frac{1}{n}\sum_{j}r_{sj} = \frac{(n+1)}{2}$ and $ \bar{r}_t=\frac{1}{n}\sum_{j}r_{tj} = \frac{(n+1)}{2}$

10. 汉明距离(hamming distance)

$$d_{st}=(\#(x_{sj}\neq x_{tj})/n)$$
两个等长字符串s1与s2之间的汉明距离定义为将其中一个变为另外一个所需要作的最小替换次数。例如字符串“1111”与“1001”之间的汉明距离为2。应用：信息编码（为了增强容错性，应使得编码间的最小汉明距离尽可能大）。

在编辑距离、拼写检查与度量空间介绍了一种hamming距离的升级版：Levenshtein距离。

1965年，俄国科学家Vladimir Levenshtein给字符串相似度做出了一个明确的定义叫做Levenshtein距离，我们通常叫它“编辑距离”。字符串A到B的编辑距离是指，只用插入、删除和替换三种操作，最少需要多少步可以把A变成B。例如，从FAME到GATE需要两步（两次替换），从GAME到ACM则需要三步（删除G和E再添加C）。Levenshtein给出了编辑距离的一般求法，就是大家都非常熟悉的经典动态规划问题。

11. 杰卡德距离(jaccard distance)

$$d_{st}=\frac{\#[(x_{sj} \neq x_{tj})\bigcap ((x_{sj} \neq 0) \bigcup x_{tj} \neq 0))]}{\#[(x_{sj}\neq 0)\bigcup (x_{tj} \neq 0)]}$$
上面这个式子很不直观，维基中给的要好理解一些：

  • 杰卡德相似系数
    两个集合A和B的交集元素在A，B的并集中所占的比例，称为两个集合的杰卡德相似系数，用符号J(A,B)表示。
    $$J(A,B) = \frac{|A\bigcap B|}{|A \bigcup B|}$$
  • 杰卡德距离
    杰卡德相似系数$J(A,B)\in [0, 1]$，定义jaccard距离：
    $$d_J(A,B) = 1-J(A,B) = \frac{|A \bigcup B|-|A\bigcap B|}{|A \bigcup B|}$$

Jaccard系数主要用于计算符号度量或布尔值度量的个体间的相似度，因为个体的特征属性都是由符号度量或者布尔值标识，因此无法衡量差异具体值的大小，只能获得“是否相同”这个结果，所以Jaccard系数只关心个体间共同具有的特征是否一致这个问题。
